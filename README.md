# Plantry – Smart Pantry & Household Management App

Plantry is a smart, fully client-side pantry and household management application designed to help users organize food and daily essentials, reduce waste, and make smarter meal and budgeting decisions.

This project focuses on combining inventory management, AI-powered suggestions, and user-friendly design into well verse application. 

---

##  Features

### Implemented 
* Allows the user to create and manage a digital pantry.
* In app warnings for expringing and expired food. 
---

## Planned

###  User Interface

* Modern, responsive UI design
* Supports **Dark Mode & Light Mode**
* Clean layout with intuitive navigation

---

### Meal Planning & Preparation

* Weekly meal prep planning system
* Generates meal plans suggestions based on available pantry items
* Automated reminders (Based on Time Zone) 

---

### AI Recipe Generator

* Generates recipes based on ingredients currently in the pantry
* Supports:

  * Ingredient substitutions
  * Missing ingredient suggestions
* Helps reduce food waste and improve meal variety

---

###  Meal Suggestion Quiz

* In-app quiz to determine what the user is in the mood for
* Suggests meals based on:

  * Preferences
  * Pantry inventory
* Provides:

  * Step-by-step cooking instructions
  * Links to external recipes or displays instructions directly

---

###  Shopping List & Bulk Add

* Create and manage shopping lists
* Bulk item entry:

  * Manual input
  * Image-based input (receipt or product images)
    
* Designed for efficient grocery shopping

---

###  Receipt Scanning

* Upload receipt images to extract purchased items
* Automatically adds items to pantry inventory
* Can be used to update budget tracking

---

### Allergy Management

* Tag and sort items by allergens
* Search ingredients based on allergy restrictions
* Helps users avoid unsafe food choices

---

###  Budget Tracker

* Track spending on groceries and household items
* Input methods:

  * Manual entry
  * Receipt scanning data
* Includes:

  * Visual graphs
  * Analytics for spending habits

---

###  Family Account System

* Create and manage family accounts
* Multiple users per household
* Each user can:

  * Add items
  * Manage inventory
* Secure login with username and password

---

### Custom Categories

* Predefined categories for items
* Users can:

  * Create custom categories
  * Assign unique colors to each category
* Supports organization beyond food (household items, supplies, etc.)

---

###  Household Item Support

* Not limited to pantry items
* Can manage:

  * Food
  * Cleaning supplies
  * General household inventory

---

## Security

* Designed with strong client-side security practices
